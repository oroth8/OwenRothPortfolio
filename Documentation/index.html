Overview

Monica is a modern HTML Template ideal for portfolio, resume or vcard. The template contains sections that will help to highlights your profile with the most relevant information.

The template was built using the latest version of the popular bootstrap framework, it has images gallery, carousel and other attractions that help to improve the user experience.

The template is totally customizable, has a clean code and very well documented for an easy understanding, also it is built with the latest web standards. It is totally responsive to all devices and it has a beautiful and modern layout.

File Structure

The structure of this template consists of HTML, CSS and JAVASCRIPT files with a good organization and very clean code. The code is well segmented and commented for a quick understanding.

    • CSS Files

    This template was built using the Bootstrap Framework, added some additional styles to customize it, below is thet list of CSS files used:

    bootstrap.css: very popular HTML, CSS, and JavaScript framework for developing responsive, mobile-first web sites.

    stylesheet.css: main custom style sheet, here you will find the most important styles of almost the entire template. It is commented on for its quick interpretation.

    main.css: contains tamplate base CSS styles.

    normalize.css: preserves useful defaults, unlike many CSS reset and normalizes styles for a wide range of elements, also corrects bugs and common browser inconsistencies.

    navbar.css: contains navbar css style.

    JAVASCRIPT Files

    This template contains some JavaScripts files, some are Bootstrap framework and others simply to create a good browsing experience, below is thet list of JAVASCRIPT files used:

    bootstrap.min.js: Bootstrap JavaScript file.

    plugins.js contains all comprest javascript.

    main.js: contains all javasscript for the template.

    nav-fixed-top.js: Navigation javascript.
